# 分布式块存储演进总览

## 什么是块存储

块存储 = 提供原始块设备（Block Device）→ 可格式化为任意文件系统 → 挂载为本地磁盘

| 维度 | 文件存储 | 块存储 | 对象存储 |
|------|----------|--------|----------|
| 接口 | POSIX 文件 API | 块设备 (/dev/sdX) | HTTP REST API |
| 粒度 | 文件 | 块 (512B-4KB) | 对象 |
| 随机读写 | ✅ | ✅ | ❌ (整体覆盖) |
| 低延迟 | ✅ | ✅ 最优 | ❌ |
| 扩展性 | 受限 | 受限 | 最优 |
| 典型用途 | 共享文件 | 数据库/VM 磁盘 | 数据湖/备份 |

## 演进脉络

```
SAN/NAS (1990s) → iSCSI/FC (2000s) → Ceph RBD (2010) → DRBD
→ CurveBS (2021) → Longhorn (2019) → SPDK/NVMe-oF (2016)
```

## 核心驱动力

1. **虚拟化**：VM 需要虚拟磁盘 → 块存储是 VM 的基础设施
2. **容器持久化**：Kubernetes PV/PVC 需要块存储后端
3. **低延迟需求**：数据库等场景需要块级随机读写
4. **软件定义存储**：从专用 SAN 硬件到纯软件方案

## 技术演进主线

### 传输协议
- Fibre Channel → iSCSI (TCP/IP) → NVMe-oF (RDMA/TCP) → SPDK (用户态)

### 架构模式
- 集中式 SAN 阵列 → 分布式软件定义 → 云原生微服务 → 高性能用户态

### 数据保护
- RAID (硬件) → 副本 (软件) → 纠删码 → 快照 + 克隆

## 时代划分

### 第一代：传统 SAN/NAS (1990s-2010)

| 技术 | 核心特征 |
|------|----------|
| Fibre Channel SAN | 专用光纤网络、高性能、高成本 |
| iSCSI | 基于 IP 网络、降低门槛 |
| NAS (NFS/SMB) | 文件级共享、易用 |

### 第二代：软件定义块存储 (2010-2018)

| 系统 | 年份 | 核心创新 |
|------|------|----------|
| Ceph RBD | 2010 | RADOS 上层、快照、克隆、KV 客户端缓存 |
| DRBD | 2000s | 网络 RAID-1、主备块复制 |
| GlusterFS (块模式) | 2010s | 通过 FUSE 提供块设备 |

### 第三代：云原生块存储 (2018+)

| 系统 | 年份 | 核心创新 |
|------|------|----------|
| CurveBS | 2021 | 网易开源、Raft、低延迟、K8s CSI |
| Longhorn | 2019 | 微服务架构、每卷独立控制器、K8s 原生 |
| AWS EBS | 2006+ | 云托管块存储、快照、多 AZ |

### 第四代：极致性能 (2016+)

| 技术 | 核心创新 |
|------|----------|
| SPDK | 用户态、零拷贝、轮询模式、无中断 |
| NVMe-oF | 通过网络暴露 NVMe 语义、RDMA 加速 |

## 核心技术对比

| 维度 | Ceph RBD | CurveBS | Longhorn | SPDK/NVMe-oF |
|------|----------|---------|----------|--------------|
| 架构 | RADOS 上层 | 独立分布式 | 微服务 per-volume | 用户态框架 |
| 共识 | CRUSH | Raft | 控制器 HA | 无 |
| 延迟 | ms 级 | 亚 ms | ms 级 | μs 级 |
| 扩展性 | 高 | 中 | 中 | 取决于网络 |
| K8s CSI | ✅ | ✅ | ✅ | 有限 |
| 适用场景 | 通用云存储 | 低延迟 OLTP | K8s 原生 | 极致性能 |

## 块存储在分布式存储中的定位

```
                    应用层
                       │
         ┌─────────────┼─────────────┐
         ↓             ↓             ↓
     文件存储       块存储       对象存储
     (CephFS)      (RBD)        (RGW)
         │             │             │
         └─────────────┼─────────────┘
                       ↓
                  统一底层
                  (RADOS / 3FS)
```
